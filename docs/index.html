text-privacy-converter

convert a text so as to protect privacy while keep format.

Getting started

  • npm
  • github
  • umdjs (for browser, etc)

Note

  • optimized for speed
  • IE11 needs polyfill (String.fromCodePoint/String.prototype.codePointAt)

Usage

const {privacyConvertEn, privacyConvertJa} = require("text-privacy-converter");
//import {privacyConvertEn, privacyConvertJa} from "text-privacy-converter"
console.log(privacyConvertJa("星空を眺めて"));//"場組わ鳥ござ"
